# ktokarska.github.io

Personal academic website of Kasia Tokarska, served via **GitHub Pages**.

- **Live site:** https://kasiatokarska.com
- **Custom domain:** configured via the `CNAME` file (`kasiatokarska.com`) and repo Settings → Pages.
- **Source:** static HTML/CSS/JS — no build step. `index.html` is the entry point; all links are relative.

## Local preview

Open `index.html` in a browser, or run a local server:

    python3 -m http.server 8000

then visit http://localhost:8000

## Deploy

Pushing to the `main` branch publishes automatically via GitHub Pages.
